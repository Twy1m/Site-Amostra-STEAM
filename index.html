O que são robôs industriais


São máquinas programáveis utilizadas principalmente

em ambientes de produção industrial para automatizar tarefas


Quais são os tipos de robô?


  • Cartesiano / Movimento X, Y, e Z
  • Cilíndrico / Movimento circulares + lineares
  • Esférico|Polar / Braço Giratório com extensão Radial
  • Articulado / Braço com várias juntas rotacionais (braço humano)
  • Scara / Rápido com 4 eixos ideal para movimento horizontal
  • Delta / Alta velocidade e precisão, ideal para pequenos objetos
